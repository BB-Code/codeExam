## 快速开始

### 步骤 1. 准备

下载仓库，安装并启动

```bash
$ git clone git@github.com:BB-Code/codeExam.git
$ cd codeExam
$ npm install
```
此时小程序编辑器打开项目，点击编译可以看到运行效果
